# Music Distribution System

Full-stack application for managing music tracks and distributing them to DSPs (Digital Service Providers) like Spotify, Apple Music, and YouTube Music.

## Tech Stack

- **Backend**: ASP.NET Core Web API (.NET 9)
- **Frontend**: Angular 18
- **Database**: SQL Server with EF Core
- **Architecture**: Clean Architecture

## Prerequisites

- .NET 9 SDK
- Node.js 18+
- SQL Server (LocalDB or full instance)
- Angular CLI
